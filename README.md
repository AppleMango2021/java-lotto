# 🚀2단계 - 문자열 덧셈 계산기

##문자열 덧셈 계산기란?
쉼표(,) 또는 콜론(:) 등을 구분자로 가지는 문자열을 전달하였을 때, 구분자를 기준으로 분리한 
각 숫자의 합을 반환하는 계산기 
(ex) "1,2" => 3

##요구사항
1. 쉼표(,) 또는 콜론(:)을 구분자로 하여 문자열을 분리할 수 있다.
2. 쉼표와 콜론 이외의 Custom 구분자를 사용하여서도 문자열을 분리할 수 있다. 
    * 이때, 커스텀 구분자는 "//"와 "\n" 사이에 위치하는 문자이다.
    * 커스텀 구분자를 사용하는 경우, 문자열 앞에 먼저 입력된다.
3. 숫자 이외의 값 또는 음수가 전달되는 경우, 예외를 던진다.
